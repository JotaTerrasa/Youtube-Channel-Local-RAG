# Changelog

## English

## 0.1.0

- First local RAG agent for YouTube videos, playlists, and channels.
- Whisper transcription in Docker with CUDA support.
- Portable Whisper CPU mode for macOS/Windows/Linux.
- Docker Compose override for NVIDIA CUDA.
- Local indexing with ChromaDB.
- LangGraph agent using Ollama with `gemma4:e2b`.
- CLI for ingestion, questions, chat, health checks, and stats.
- Transcript cache and support for `--skip-cached`.
