# Changelog

## 0.1.0

- Primer agente RAG local para videos, playlists y canales de YouTube.
- Transcripción con Whisper en Docker y CUDA.
- Modo Whisper CPU portable para macOS/Windows/Linux.
- Override Docker Compose para NVIDIA CUDA.
- Indexado local con ChromaDB.
- Agente LangGraph usando Ollama con `gemma4:e2b`.
- CLI para ingesta, preguntas, chat, estado y estadísticas.
- Cache de transcripciones y soporte para `--skip-cached`.
